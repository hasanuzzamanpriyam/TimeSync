
Create a production-ready desktop Task Tracking Application inspired by Trackabi with modern UI/UX and enterprise-level architecture.

PROJECT GOAL
Build a cross-platform desktop application that runs on:
- Windows
- Linux
- macOS

The application will be integrated with an ERP system through REST APIs.

The system should allow employees to track tasks, time, productivity, attendance, and work logs while synchronizing data with the ERP backend.

TECHNOLOGY STACK

Desktop Application:
- Tauri + React + TypeScript (preferred)
- Alternative: Electron + React + TypeScript

Frontend:
- React
- TypeScript
- Vite
- React Router
- Axios
- Zustand or Redux Toolkit
- Tailwind CSS
- ShadCN UI

Local Database:
- SQLite

Backend Communication:
- REST API
- JWT Authentication
- Refresh Token Support

Build Targets:
- Windows (.exe)
- Linux (.AppImage/.deb)
- macOS (.dmg)

GET /api/tasks
GET /api/projects
POST /api/time-entries
POST /api/attendance

- Retry failed requests

- Sync manager
